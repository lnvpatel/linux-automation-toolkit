# Repository Architecture

This repository is organized into operational categories:

* `monitoring/` → system monitoring scripts
* `networking/` → network diagnostics and port inspection
* `backup/` → backup automation scripts
* `cleanup/` → maintenance and cleanup automation
* `user-management/` → Linux user administration scripts
* `docs/` → operational notes and documentation

The goal of this project is to build practical Linux administration and infrastructure automation scripts with production-style structure and troubleshooting awareness.
