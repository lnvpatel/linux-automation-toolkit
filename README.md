# Linux Automation Toolkit

Linux automation toolkit focused on system administration, monitoring, networking, backup automation, cleanup operations, and user management.

This project was built to improve practical Linux administration and infrastructure automation skills through operational scripting and troubleshooting-focused workflows.

---

# Features

## System Monitoring

* Disk usage monitoring
* Memory monitoring
* CPU monitoring

## Networking

* Network health checks
* Port and service inspection
* Connectivity diagnostics

## Backup Automation

* File and directory backup automation
* Database backup workflow
* Timestamped compressed archives

## Cleanup Operations

* Log cleanup automation
* Temporary file cleanup

## User Management

* Linux user creation automation
* User account disable operations

## Documentation

* Troubleshooting notes
* Linux command references
* Script usage guides
* Repository architecture documentation
